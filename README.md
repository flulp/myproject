# Калькулятор

1. Калькулятор.
2. Производит вычислительные операции с числами такие как:
* Сложение(+)
* Вычитание(-)
* Умножение(*)
* Деление(/). 
3. Запуск программы:
- В консоле вводим "javac Calculator.java"
- Вводим в консоль "java Calculator"
4. Способ работы:
- Введите 1 число
- Введите 2 число
- Выберите пункт из предложенных
